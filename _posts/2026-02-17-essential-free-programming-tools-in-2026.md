## Code Editor – Visual Studio Code

A code editor is where you write, edit, and manage your code.  
For beginners, **Visual Studio Code (VS Code)** remains the best all-around choice.

### Why beginners love it

- **Simple interface** – You can start coding immediately without configuration chaos.
- **Beginner-friendly defaults** – It works well right after installation.
- **Huge community support** – Almost every issue already has a solution online.
- **Cross-platform** – Same experience on Linux, Windows, and macOS.

### What you can do with it

- Write code in Python, JavaScript, HTML, CSS, C, Java, and more  
- Debug programs by pausing execution and inspecting variables  
- Install extensions for formatting, linting, and automation  
- Use Git directly inside the editor  

👉 Official download: <a href = "https://code.visualstudio.com/" target = "_blank" rel = "noopener noreferrer">https://code.visualstudio.com/</a>

## Version Control – Git & GitHub

Version control tracks changes in your code over time.

If coding were a video game, **Git** would be the save system.

### Why beginners should use Git early

- **Undo mistakes** – Roll back to a working version.
- **Industry standard workflow** – Used in companies and open source.
- **Build a public portfolio** – GitHub shows your progress to employers.

### Git vs GitHub (Simple Comparison)

| Tool   | Purpose |
|--------|----------|
| Git    | Tracks changes locally on your computer |
| GitHub | Hosts your Git projects online |

👉 Install Git: <a href = "https://git-scm.com/" target = "_blank" rel = "noopener noreferrer">https://git-scm.com</a>

👉 Create a GitHub account: <a href = "https://github.com" target = "_blank" rel = "noopener noreferrer">https://github.com</a>

## Terminal / Command Line (CLI)

The terminal lets you communicate directly with your operating system.

It looks intimidating at first, but it’s one of the most powerful tools you’ll ever use.

### Why the terminal is essential

- **Faster than clicking around** – Commands replace multiple mouse actions.
- **Required for dev tools** – Git, package managers, and servers rely on it.
- **Helps you understand your system** – You learn how files and processes work.

### Terminal Options (Comparison Table)

| Tool                    | OS       | RAM Usage | Features                     | Beginner Friendly |
|-------------------------|----------|-----------|------------------------------|-------------------|
| Default Linux Terminal  | Linux    | Very Low  | Fast, stable                 | ✅ |
| Windows Terminal        | Windows  | Low       | Tabs, themes, Unicode        | ✅ |
| Git Bash                | Windows  | Low       | Unix-like commands           | ✅ |

👉 Windows Terminal: <a href = "https://aka.ms/terminal" target = "_blank" rel = "noopener noreferrer">https://aka.ms/terminal</a>

## Programming Language Runtime – Python

You can’t run code without installing the language runtime.

For beginners in 2026, **Python** is still the best starting point.

- **Readable syntax** – Looks close to plain English.
- **Huge ecosystem** – Web, automation, data, AI, cybersecurity.
- **Clear error messages** – Easier to debug than many languages.
